## 6. Core Gap Handling Strategy

When a Core public API is missing:

1. **Do NOT** implement the missing capability in iOS.
2. **Do NOT** copy Core internal logic.
3. **Document** the gap in this plan with a handoff tag.
4. **Mark** the dependent iOS task as `BLOCKED` or `MOVED_TO_CORE`.
5. **Proceed** with iOS work that does not depend on the gap.
